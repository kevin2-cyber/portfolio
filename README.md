# Kelvin Eduful — Portfolio

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a dark/light theme, smooth animations, and a fully responsive layout.

## Live Demo

> Deploy via GitHub Pages: `Settings → Pages → Branch: main → / (root)`

## Features

- Dark / Light mode toggle (persisted in `localStorage`)
- Animated typing effect in the hero section
- Scroll-triggered fade-in animations
- Animated stat counters
- Mobile-responsive with hamburger menu
- Contact form with client-side validation
- Smooth scroll with active nav link tracking

## Project Structure

```
portfolio/
├── index.html        # Main entry point
├── css/
│   └── style.css     # All styles & CSS variables
├── js/
│   └── script.js     # Interactivity & animations
└── html/
    ├── about.html
    └── contact.html
```

## Getting Started

Open `index.html` directly in a browser — no build step required.

```bash
# Or serve locally with any static server, e.g.:
npx serve .
```
